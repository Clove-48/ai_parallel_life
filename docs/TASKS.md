| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|----------|
| LLM API 响应不稳定 | 中 | 高 | 实现超时重试、提供降级文案 |
| AI 叙事质量不达标 | 中 | 高 | Prompt 多轮调优、预设示例输出 |
| 移动端兼容性问题 | 低 | 中 | 主流浏览器适配测试 |
| 后端部署环境差异 | 低 | 低 | Docker 容器化部署 |
